# 易管家（Yi-housekeeper）
## 项目描述
- 基于CSharp的winform框架开发；
- 项目实现了检查更新、资源监控、垃圾清理、文件扫描等功能
- 目前电脑体检、开机加速、软件管家功能有待实现
- 模仿腾讯电脑管家的简单软件
